============================================================
 即興カスタム チームメイカー を「いつでも開けるURL」にする
 （クラウド常設・無料・クレジットカード不要 / Render を使用）
============================================================

あなたの作業はこれだけ。順番にやればOKです。

------------------------------------------------------------
 ステップ A：コードを GitHub に上げる
------------------------------------------------------------
1. このフォルダの「upload-to-github.bat」をダブルクリック
2. 画面の案内どおり：
   - https://github.com/new を開く
   - Repository name に「cd6-customs」と入力（名前は何でもOK）
   - 「Public」を選ぶ（README等のチェックは付けない）
   - 「Create repository」をクリック
   - 表示されたURL（例 https://github.com/ユーザー名/cd6-customs.git）をコピー
3. bat画面にそのURLを貼り付けて Enter
   → 途中でブラウザが開いたら GitHub にサインイン（1回だけ）
   → 「DONE!」と出れば完了

------------------------------------------------------------
 ステップ B：Render で公開する
------------------------------------------------------------
1. https://render.com を開く
2. 「Get Started」→「GitHub」でサインイン（ステップAと同じGitHubでOK）
3. 画面右上「New +」→「Blueprint」を選ぶ
4. さきほどの「cd6-customs」リポジトリを選んで接続
5. 「Apply」(または Create) をクリック
   → 自動でビルドが始まります（2〜3分）
6. 完成すると、次のようなURLが発行されます：
        https://cd6-customs.onrender.com
   これが「いつでも開けるURL」です。
   開く → 「QR生成してスタート」→ トーナメント作成まで全部できます。

============================================================
 （任意）独自ドメイン customs.cd6.io で開けるようにする
============================================================
1. Render のサービス画面 → 「Settings」→「Custom Domains」
2. 「customs.cd6.io」を追加 → 表示される宛先（cd6-customs.onrender.com）を控える
3. Cloudflare の cd6.io の DNS に CNAME を追加：
     Name=customs  /  Target=cd6-customs.onrender.com  /  Proxy=DNS only(グレー)
4. しばらく待つと https://customs.cd6.io で開けるようになります

============================================================
 知っておくこと（無料プランの仕様）
============================================================
・15分アクセスが無いとスリープし、次に開くとき起動に30〜60秒かかります
  （イベント中は管理画面が自動通信するのでスリープしません）
・スリープ/再起動すると保存データはリセットされます
  → 毎回まっさらな状態で始められるので、即興カスタムには好都合です
・完全無料・クレジットカード不要です
